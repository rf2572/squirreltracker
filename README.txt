
This project is meant to help squirrel lovers track squirrels. The sightings application allows user to import squirrel data from a csv file and display it on an html page. Different views give viewers the ability to add each sighting information, add new sightings and view stats or maps on current sightings. Management command also allows user to export data from their database.
